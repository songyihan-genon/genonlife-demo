# 콜 데모 시연 시나리오 - 상담사 계정 v.1.0

### 데모 시연 흐름

`김민준 케이스-실시간 상담 및 후처리: [AI홈]→콜받기→[실시간상담]→[SMS]→[상담이력조회]→[접촉이력등록]→반영확인 
정해린 케이스- AI 검수 및 2차 검수: [상담이력조회] 검토완료 필터→[검수결과]`

## STEP 1 — [AI 상담 홈] 대시보드 및 실시간 콜 인입

!image.png

**핵심 기능**: 상담사 KPI 대시보드 + 실시간 콜 수신 + 고객정보 자동 로딩 + 최근상담 이력 

- **보여줄 것**: 진입 후 약 3초 → "수신 대기 → **인입 중**"으로 전환(점 깜빡임). 약 1초 뒤 고객(**김민준**) 정보 카드가 자동 로딩.
- **조작**: **[통화 받기]** 클릭 → 실시간 상담 화면 진입.
- **멘트**: "콜이 들어오면 누가 거는지, 어떤 이력의 고객인지 받기 전에 이미 떠 있습니다."

## STEP 2 — [실시간 고객 상담] 상담 중 일어나는 일

!image.png

**발화 주제 흐름**:

| 발화 주제(순서) | 주목할 애니메이션·반응 |
| --- | --- |
| ① 도입·본인확인 | 상단 **컨텍스트 바**에 고객정보 + 이력 **플래그 3종**(실효 이력·안내장 수령·보험금 접수중) 등장 / **FCR 필수 안내** 항목이 에메랄드 펄스로 자동 체크 |
| ② 보험금 청구 **추가서류**  | 발화 속 키워드 **노란 하이라이트** → 가이드에 키워드칩 등록 후 **파란 펄스 자동 선택** / **RAG** 질문 전송 → 답변(요약·근거·출처) / **업무정보**에  청구 서류 **제출 현황 확인** |
| ③자동이체 **계좌 변경**  | 키워드 추출· RAG(변경 기준) 검색 확인  |
| ④ **해지환급금** / 문자발송 요청 | 키워드·RAG(안내 기준) / **🔑 관리자 코칭 패널이 자동으로 펼쳐지며 '실시간 청취 중'** + 코칭 메시지 도착(해지 고지 유의) / 업무정보 내 가입상품-종신보험 확인 |
| 종료 | 통화 종료 → 헤더 '내 상태' **통화 대기** 자동 전환 / 우측 **'SMS 안내 발송'** 카드 점등 |

## STEP 3 — [SMS 문자 안내 작성]

!image.png

**핵심 기능**: 상담 요약 기반 문자 초안 생성·편집·발송을 한 화면에서 처리

- **보여줄 것**: 톤 칩 선택 → **[초안 생성]**(생성 전 화살표가 깜빡이며 유도, 1회 생성 후 멈춤) → 좌측 근거 기반 문자 초안이 채워짐. / 톤 조정 후 재생성 가능
- **조작**: 초안 확인 → **[발송하기]** → "발송 중…" → **[발송완료]**(파랑 유지), 편집창 비활성화 + *"김민준 고객님 000-000-0000로 발송되었습니다"*.

---

## STEP 4 — [상담 이력 조회]

!image.png

**핵심 기능**: 상담 테이블 탐색 + 고객 별 상담 상세 카드 + 후속업무 상태 연동 / 중간에 콜인입 배너 등장 

- **보여줄 것**: 기본 **오늘** 테이블, 김민준 행: **SMS = 발송완료**(방금 발송 반영), **접촉이력 = 미등록**(아직 미처리).
- **조작**: 김민준 행 클릭 (또는 **'미등록'** 셀 클릭) → 김민준 고객 카드 전체 리뷰 후 접촉이력 등록 버튼

---

## STEP 5 — [접촉이력 등록]

!image.png

**핵심 기능**: 종료 상담 자동 분석 → 유형 분류·요약 초안 → 등록

- **화면 구성**: 3분할 — **상담 개요 / 접촉 유형 분류 / 접촉 이력**.
- 보여줄것: 유형별 대분류/중분류/소분류 탭·요약 초안 자동 작성. 해당 부분 발화시점 확인가능. 유형 추가 가능
- **조작**: **[접촉이력 등록]** 클릭 → **등록 완료**로 전환되며, 등록 내용이 **상담 개요에 반영**(반영 로딩 후 갱신).

## STEP 7 — 검토 완료 필터 → (정해린) AI 검수 결과 보러가기

!image.png

**핵심 기능**: 검수 상태 필터 + 감지 건 바로가기

- **조작**: 테이블 상단 검수 필터에서 **'검수 완료'** 선택(AI 일괄 검수가 끝난 과거 건만 표시).
- **보여줄 것**: **정해린**(CL-20260513-027) 행 — AI 검수 = **'오안내 1·누락 1'** 태그 + **심각** 배지.
- **주목 애니메이션**: **AI 검수 셀에 호버 → 화살표(→)** → 클릭하면 검수 결과로 바로 이동. (행 클릭 후 상세 카드의 **[결과 보기 →]** 로도 진입 가능)

## STEP 8 — [상담 검수 결과 화면]

!image.png

**핵심 기능**: AI 1차 검수 + 휴먼(관리자) 2차 검수를 한 화면에

- **AI 검수 종합**: 환급률 오안내 + 보장 종료 고지 누락 감지(심각) 요약 + 원문 확인 가능
- **탭**: **[오안내 검수] / [안내 누락 검수]** — 각 탭에 판정 카드(평가 사유) + 판정 근거(약관·내규) 원문보기 + 매뉴얼 수행 여부.
- **휴먼 검수**: 상태 **조치 필요**, 관리자 코멘트 + 조치 옵션(**고객 재안내 전화 / 정정 문자 재발송 / 상담사 재교육**).
