# GenON LIFE AICC — 데모 시연 플레이북

> 단일 시연 흐름. 각 화면의 **패널·카드 구성(어떤 기능이 있는지)** 을 먼저 설명하고, 그 위에서 **무엇을 조작/주목**하는지 정리합니다.

**흐름**: `[AI 상담 홈] → (김민준) 콜 받기 → [실시간 상담] → [SMS 안내] → [상담 이력 조회] → [접촉이력 등록] → 이력 반영 확인 → 검수 완료 필터 → (정해린) AI 검수 결과 → [검수 결과 화면]`

---

## STEP 1 — [AI 상담 홈]

상담사 업무의 출발점. 좌측은 **개인 대시보드 + 업무 어시스턴트**, 우측은 **실시간 콜/이력** 사이드바로 구성됩니다.

**화면 구성 (패널·카드)**
- **상담 KPI 대시보드**(상단): `오늘의 상담 목표율`(14/60 진행바), 처리 건수·고객만족도 등 KPI, `후처리 대기 N건`(앰버 표시) — 오늘 내 업무 현황을 한눈에.
- **나만의 업무 어시스턴트**: 그라데이션 테두리 검색창(약관·규정·스크립트 검색) + 예시 질문 리스트.
- **바로가기**: 자주 쓰는 업무 화면 4종 카드(아이콘 그리드).
- **콜 인입 패널**(우측 상단): 수신 대기 ↔ 인입 중 상태. 인입 시 `대기 콜·대기 시간·채널` 수치, **[통화 받기]** 버튼, **고객 정보 카드**(고객명/번호/등급/채널 + **AI 예상 문의**).
- **오늘의 고객 상담 이력**(우측 하단): 최근 상담 목록(고객·요약·상담번호·시간 + `후처리 필요` 태그). 클릭 시 해당 후속업무 화면으로 이동.

**조작 / 주목**
- 진입 ~3초 후 콜 인입(점 깜빡임) → ~1초 뒤 **고객(김민준) 정보 자동 로딩**(AI 예상 문의 포함).
- **[통화 받기]** 클릭 → 실시간 상담 화면.
- 💬 "콜을 받기도 전에 누가, 어떤 이력의 고객인지, 무엇을 물어볼지까지 AI가 미리 띄워줍니다."

---

## STEP 2 — [실시간 고객 상담]

상단 컨텍스트 바 + 4분할 작업 공간. **STT·가이드·지식검색(RAG)·업무정보**가 발화에 맞춰 동시에 반응합니다.

**화면 구성 (패널·카드)**
- **컨텍스트 바**(상단): 고객·상담ID + 이력 **플래그**(실효 이력·갱신 안내장·보험금 접수중).
- **① STT 패널**: 화자별 말풍선 실시간 표시. 고객 발화의 **키워드 노란 하이라이트**, 말풍선 다중선택 → 복합 질의.
- **② 상담 가이드 패널**: `FCR 필수 안내 확인`(본인확인·녹취고지 등 자동 체크), `추천 상담 스크립트`(대화 유형별), `대화 키워드·FAQ`(추출 키워드 + 추천 질의).
- **③ 상담지식 어시스턴트(RAG) 패널**: 약관·업무 기준 **지식 질의** 전용. `질문 → 탐색 4단계 → 답변(요약·근거·출처·업무기준)` + FAQ 칩 + 입력창.
- **④ 우측 패널(상하 분할)**:
  - **업무정보**: 계약 선택 + **처리계 연동 데이터**(청구 서류 제출 현황·처리 가능 여부·예상 환급금/한도·유의사항).
  - **Next Step·작업 추천**: 토픽별 다음 업무 화면 + 후속 작업(SMS 안내·접촉이력 등록).
- **관리자 코칭 패널**(우측 하단): 관리자 실시간 청취·코칭 메신저.

> 💡 역할 분리: **RAG = 기준/약관**, **업무정보 = 고객의 실제 현황**(제출/미제출, 처리 가능 여부 등 연동 데이터).

**발화 주제 흐름 & 주목 애니메이션** *(자동재생 — 별도 조작 불필요)*

| 발화 주제 | 주목할 애니메이션 |
| --- | --- |
| 도입·본인확인 | 컨텍스트 바 플래그 3종 등장 · FCR 항목 **에메랄드 펄스 자동 체크** |
| 보험금 청구 **추가서류** | 키워드 **노란 하이라이트** → 키워드칩 **파란 펄스 자동 선택** · RAG **탐색 4단계 → 답변** · 업무정보에 청구 서류 **제출 현황** 표시 |
| 세부내역서 안내 | Next Step 카드 강조 + 후속작업 추천 점등 |
| 자동이체 **계좌 변경** | RAG(변경 기준) · 업무정보가 종신 계약으로 전환 + 처리 가능 여부 |
| 처리 가능 확인 | 업무정보 **유의사항** 강조 |
| **해지환급금** | RAG(안내 기준) · **관리자 코칭 패널 자동 펼침 + '실시간 청취 중' + 코칭 메시지 도착** |
| 통화 종료 | '내 상태' **통화 대기** 자동 전환 · **'SMS 안내 발송'** 카드 점등 |

- STT가 길어지면 **자동 스크롤업**.
- **조작**: 점등된 **'SMS 안내 발송'** 클릭 → SMS 화면.

---

## STEP 3 — [SMS 안내 작성]

상담 내용을 근거로 안내 문자를 만들어 발송하는 3분할 화면.

**화면 구성 (패널·카드)**
- **좌: 근거 패널** — 어시스턴트가 정리한 안내 근거 + 추천 질의 칩(해지환급금 산출 근거 등).
- **가운데: 초안 생성 브리지** — `좌측 근거 → 초안 생성` 변환 버튼(화살표).
- **우: 문자 안내 초안 패널** — '어시스턴트 근거 기반 · 상담사 편집' 가능한 초안 + **[발송하기]**.

**조작 / 주목**
- **[초안 생성]**: 생성 전 화살표가 깜빡이며 유도(1회 생성 후 멈춤) → 근거 기반 초안 자동 작성.
- **[발송하기]** → "발송 중…" → **[발송완료]**(파랑 유지)·편집창 비활성 + *"김민준 고객님 000-000-0000로 발송되었습니다"*.

---

## STEP 4 — [상담 이력 조회]

상담 테이블 탐색 + 행 선택 시 우측 상세 카드.

**화면 구성 (패널·카드)**
- **필터 바**: 기간(최근 7일 등), **검수 필터(전체/검수 대기/검수 완료)**, 통화별·고객별 뷰, 검색창.
- **상담 테이블**: 상담일시·고객·채널·요약 + **접촉이력 / SMS / AI 검수 / 휴먼 검수** 상태 컬럼.
- **상세 카드(우측 드로어)** — 행 클릭 시: `가입 상품` · `상담 요약`(대화 키워드·원문 보기·요약·관리자 코칭 이력·다음 예상 문의) · `후속 작업`(SMS·접촉이력) · `고객 상태·필요 조치` · `동일 고객 상담 이력` · `상담 내용 검수`(결과 보기).

**조작 / 주목**
- 김민준 행: **SMS = 발송완료**(방금 반영), **접촉이력 = 미등록**.
- 🔑 미처리 셀(미등록/발송요청/감지)에 **호버하면 화살표(→)** → 클릭 시 해당 처리 화면으로 바로 이동.
- **조작**: 김민준 **'미등록'** 셀 클릭 → 접촉이력 등록.

---

## STEP 5 — [접촉이력 등록]

종료 상담을 분석해 유형 분류·요약 초안을 만들고 이력으로 등록하는 3분할 화면.

**화면 구성 (패널·카드)**
- **상담 개요**(좌): 전체 요약 + 유형별 요약 + 원문 통화 내용 보기.
- **접촉 유형 분류**(중): 의도 분석 기반 유형 탭 + 유형별 **요약 초안 편집**.
- **접촉 이력**(우): Tele-Pro 형식 접촉이력 입력/등록.

**조작 / 주목**
- 진입 시 **분석 연출**(요약·유형 초안 단계적 생성).
- **[접촉이력 등록]** → **등록 완료** 전환 + 등록 내용이 **상담 개요에 반영**(반영 로딩 후 갱신).

---

## STEP 6 — 상담 이력 반영 확인
- **조작**: 다시 **[상담 이력 조회]** 진입.
- **주목**: 김민준 행 접촉이력이 **등록완료**로 갱신(SMS 발송완료와 함께 후속업무 처리 완료). 💬 "처리 결과가 이력에 즉시 반영됩니다."

---

## STEP 7 — 검수 완료 필터 → (정해린) AI 검수 결과 보러가기
- **조작**: 필터 바에서 **'검수 완료'** 선택 → AI 일괄 검수가 끝난 과거 건만 표시.
- **주목**: **정해린**(CL-20260513-027) 행 — AI 검수 = **'오안내 1·누락 1'** 태그 + **심각** 배지. 셀 **호버 → 화살표 → 클릭**(또는 행 클릭 후 상세 카드 **[결과 보기 →]**) → 검수 결과 화면.

---

## STEP 8 — [상담 검수 결과]

AI 1차 검수와 휴먼(관리자) 2차 검수를 한 화면에 담은 검수 리포트.

**화면 구성 (패널·카드)**
- **고객 정보 헤더**(상단): 정해린 · 고객번호 · 상담일시 + **검토자 박관리** + **관리자 2차 검토 완료** 배지.
- **좌: 상담 대화 패널** — 검수 대상 통화 원문(화자별).
- **우: 검수 결과 패널**
  - **AI 검수 종합**: 감지 요약(환급률 오안내 + 보장 종료 고지 누락 / 심각).
  - **탭**: `오안내 검수` / `안내 누락 검수` — 탭별 **판정 카드(평가 사유)** + **판정 근거(약관·내규)** + **매뉴얼 수행 여부**.
  - **휴먼 검수**(2분할): 좌 `관리자 코멘트`, 우 `조치 상태 + 옵션`(조치 필요 → **고객 재안내 전화 / 정정 문자 재발송 / 상담사 재교육**).

**멘트**
- 💬 "AI가 감지하고(오안내·누락), 관리자가 약관·내규 근거로 확인해 재안내·재교육 조치까지 — 검수 루프가 한 화면에서 닫힙니다."
